Discuss the scope of var, let, and const

var একটিই ফাংশন স্কোপ এবং ফাংশন হোইস্টিং। let এবং const হলো ব্লোক স্কোপ। একটি ভেরিয়েবল আবার যদি ব্যবহার করতে হয় তাহলে let ব্যবহার করতে হয় এবং যখন ভেরিয়েবলটি মান স্থির অর্থাৎ কখনো চেঞ্জ হবে না তখন const ব্যবহার করতে হয়।

Tell us the use cases of null and undefined

null ব্যবহার করা হয় ভেবে চিন্তে যখন একটি ভেরিয়েবল এর মান আমরা প্রথমে ফাকা রাখবো এবং পরে গিয়ে সেটার একটি মান সেট করবো। সেক্ষেত্রে আমরা মুলত null ব্যবহার করে থাকি আর undefined হলো কোন ভেরিয়েবল এর মান আমাদের নিজেদের অজান্তে দিতে ভুলে গেছি। দুইটাই কোণ মান বহন করে না কিন্তু সামান্য এতটুকুই পার্থক্য

What do you mean by REST API?

REST API এর ফুল ফর্ম কলো Representational State Transfer Application Programming Interface। REST API কোন আলাদা প্রোগ্রামিং ল্যাংগুয়েজ না। এটা হলো একটি নর্দিষ্ট তথ্য আদান প্রদানের একটি মাধ্যম। এটি সব চেয়ে বেশি ব্যহবার হয় web applicaion এবং website এ।
